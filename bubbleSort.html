Bubble Sort




How does this work?


Bubble Sort is a fairly simple algorithm, both to implement and to understand. It works by starting with the first two items in the array, and if the first item is greater than the second, a swap is made. It repeats this process with the second and third element, contunuing until it reaches the end of the array. Once that happens, the algorithm checks if the array is fully sorted. If not, it repeats the whole process, making it quite inefficient for large arrays.



Time Complexity


Worst Case: O(n^2)


Best Case: O(n) -- (When it is already sorted)


Average Case: O(n^2)
