Selection Sort




How does this work?


Selection Sort is another simple sorting algorithm. It works by starting at either the first or last element of the array, and then checking every element after it to find the minimum. Once it loops through the rest of the array and locates it, a swap is made with the starting element and the minimum element. The algorithm then increments the location of the starting index, and repeats the process. This happens until the array is completely sorted. While this algorithm also implements two loops, it is typically faster than Bubble Sort.



Time Complexity


Worst Case: O(n^2)


Best Case: O(n^2)


Average Case: O(n^2)
