Insertion Sort




How does this work?


Insertion Sort works by starting with the second element in the array, and then runs through a loop which checks every element behind the starting index. If an element behind the starting index is greater than the starting index, the entire array is shifted to the right until the program detects that a number isn't greater anymore. This process continues on until the entire array is sorted. In its best case, this algorithm is significantly faster than selection sort.



Time Complexity


Worst Case: O(n^2)


Best Case: O(n)


Average Case: O(n^2)
