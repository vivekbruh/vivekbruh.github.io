Merge Sort




How does this work?


Merge Sort, along with Binary Search, are commonly known as "Divide and Conquer" algorithms. This means they divide up the work as they complete their respective tasks. Merge Sort works by splitting the array into two parts continuously, while simultaneously sorting them. This algorithm is coded recursively, which means a function is called within itself. This is because as the array sections split, a merge function is called repeatedly to combine each section and create the sorted array.



Time Complexity


Worst Case: O(n * log(n))


Best Case: O(n * log(n))


Average Case: O(n * log(n))
